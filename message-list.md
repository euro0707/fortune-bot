# LINE Bot メッセージ一覧

## 1. 基本診断（FlexMessage）
**入力**: `YYYY-MM-DD` 形式の誕生日
**例**: `1982-09-02`

**FlexMessage構成**:
- タイトル: 性格タイプ名（例: "純粋な理想家"）
- キャッチコピー: タイプの特徴説明
- 特徴: 性格の要約
- 弱点: 注意すべき点
- 対策: 改善提案

## 2. 詳細診断（テキスト）
**入力**: `詳しく YYYY-MM-DD`
**例**: `詳しく 1982-09-02`

**内容**:
- 人生サイクル（年周期）
- 人生の基盤
- 潜在能力
- 内面・外面・仕事スタイルの詳細分析

## 3. 相性診断（テキスト）
**入力**: `相性 YYYY-MM-DD YYYY-MM-DD`
**例**: `相性 1982-09-02 1993-10-09`

**内容**:
- 両者のタイプ表示
- ビジネス相性（星評価・数値）
- 恋愛相性（星評価・数値）
- 詳細スコア（双方向）

## 4. ヘルプメッセージ（テキスト）
**入力**: `ヘルプ`, `help`, `使い方`, `？`, `?`

**内容**:
- 使い方ガイド
- コマンド一覧
- 機能説明

## 5. 初回案内メッセージ（テキスト）
**入力**: 無効な形式の文字列

**内容**:
- Bot紹介
- 入力形式説明
- 使い方のヒント

## 6. エラーメッセージ（テキスト）
**発生条件**: システムエラー時

**内容**:
- 一般的なエラーメッセージ
- 再試行の案内

---

## 技術的特徴

### FlexMessage
- 絵文字なし（LINE API互換性のため）
- 句読点を英字に変換（。→., 、→,）
- 色指定: タイトル=#5D2E8C, 本文=#555555

### テキストメッセージ
- 絵文字使用可能
- 日本語句読点使用
- マークダウン風フォーマット

### フォールバック機能
- FlexMessage生成エラー時は自動的にテキスト版に切り替え
- 全ての機能でエラーハンドリング実装
